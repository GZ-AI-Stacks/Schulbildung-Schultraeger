1 Überblick — Was ist der LehrplanManager?

Der LehrplanManager ist Ihr persönliches Lehrer-Dashboard, mit dem Sie aus 110 Lernmodulen in 11 Fach-Apps einen individuellen Lehrplan zusammenstellen können — und diesen als fertige, eigenständige Schüler-App exportieren.

📚
110 Module in 11 Apps
Deutsch, Mathematik, Informatik, Physik, Chemie, Biologie und Englisch — für Klasse 1 bis 10.
📅
Wochenplaner
Module per Drag & Drop auf Wochentage verteilen. Mehrere Pläne für verschiedene Klassen.
🎓
Schüler-App Generator
Eine HTML-Datei zum Download — mit genau den Modulen, die Sie ausgewählt haben. Inkl. Deadline-Countdown.
🔒
100% Offline & DSGVO
Kein Server, keine Registrierung, kein Tracking. Alles läuft lokal in Ihrem Browser.
Wichtig: Der LehrplanManager ist eine einzelne HTML-Datei. Sie brauchen nur einen Browser (Chrome, Firefox, Edge, Safari) — keine Installation, keinen Account, kein Internet.

2 Schnellstart — In 5 Minuten loslegen

Dashboard öffnen

Öffnen Sie die Datei Dashboard/index.html in Ihrem Browser. Doppelklick auf die Datei genügt.

Module auswählen

Klicken Sie auf die Module, die Ihre Schüler bearbeiten sollen. Ein Haken erscheint bei jedem ausgewählten Modul. Die Statistik oben zählt automatisch mit.

Optional: Wochenplan erstellen

Wechseln Sie zum Wochenplaner (linke Seitenleiste) und ziehen Sie die Module per Drag & Drop auf die gewünschten Wochentage.

Schüler-App generieren

Klicken Sie auf den grünen Button «Schüler-App» in der Kopfzeile. Geben Sie Klassenname, Begrüßungstext und ein Abgabedatum ein. Klicken Sie auf «App herunterladen».

An Schüler verteilen

Geben Sie die heruntergeladene HTML-Datei zusammen mit dem Schulbildung-Ordner an Ihre Schüler weiter (USB-Stick, Schulcloud, E-Mail). Die Schüler öffnen die Datei im Browser und arbeiten ihre Module ab.

Fertig! In diesen 5 Schritten haben Sie einen kompletten, personalisierten Lernplan erstellt und als App an Ihre Klasse verteilt. Alles Weitere auf dieser Seite erklärt die Feinheiten.

4 Module auswählen & Lehrplan zusammenstellen

Einzelne Module auswählen

Klicken Sie auf ein Modul

Jedes Modul zeigt einen Titel, ein Symbol und die geschätzte Zeit (25 Min.). Beim Klicken erscheint ein grüner Haken ✓ — das Modul ist jetzt in Ihrem Lehrplan.

Erneut klicken zum Abwählen

Ein zweiter Klick entfernt das Modul wieder aus der Auswahl. Der Haken verschwindet.

Ganze Apps auf einmal auswählen

Am unteren Rand jeder App-Karte finden Sie den Link «Alle auswählen». Damit werden alle 10 Module dieser App auf einmal ausgewählt. Bei erneutem Klick werden alle wieder abgewählt.

Schnellzugriff (Seitenleiste)

AktionBeschreibung
Alle auswählenWählt alle 110 Module in allen 11 Apps auf einmal aus.
Auswahl leerenEntfernt alle Auswahlen und leert auch den Wochenplan.

Nach Fächern oder Klassenstufen filtern

Nutzen Sie die Filter-Chips oberhalb der App-Karten:

  • Fach-Filter: Deutsch, Mathematik, Informatik, Physik, Chemie, Biologie, Englisch
  • Klassen-Filter: Kl. 1-3, Kl. 3-6, Kl. 7-10

Es wird immer nur ein Filter gleichzeitig aktiv. Klicken Sie auf «Alle Fächer», um den Filter zurückzusetzen.

Tipp: Ihre Auswahl bleibt erhalten, auch wenn Sie filtern. Sie können z.B. zuerst Mathe-Module auswählen, dann auf Deutsch wechseln und weitere Module hinzufügen.

5 Der Wochenplaner

Im Wochenplaner verteilen Sie Ihre ausgewählten Module auf die fünf Schultage Montag bis Freitag.

Module zuweisen

Wechseln Sie zum Wochenplaner

Klicken Sie in der Seitenleiste auf «Wochenplaner».

Module per Drag & Drop zuweisen

Unter dem Wochenplan sehen Sie alle ausgewählten Module. Ziehen Sie ein Modul mit der Maus auf den gewünschten Wochentag und lassen Sie los.

Module zwischen Tagen verschieben

Bereits zugewiesene Module können Sie von einem Tag auf einen anderen ziehen. So passen Sie Ihren Plan flexibel an.

Module aus einem Tag entfernen

Klicken Sie auf das × rechts neben dem Modul im Wochenplan.

Hinweis: Der Wochenplan zeigt unten bei jedem Tag die Gesamtzeit an (z.B. «75 Min.» bei 3 Modulen). So sehen Sie sofort, ob ein Tag zu voll wird.

Mehrere Lehrpläne verwalten

Sie können beliebig viele Lehrpläne anlegen — z.B. einen pro Klasse oder pro Woche:

AktionWie?
Neuen Plan anlegenKlicken Sie auf «+ Neuer Plan» und geben Sie einen Namen ein (z.B. «7a MINT-Woche»).
Zwischen Plänen wechselnKlicken Sie auf den jeweiligen Plan-Tab.
Plan umbenennenKlicken Sie auf das Stift-Symbol ✏ neben dem Plannamen.
Plan löschenKlicken Sie auf das × neben dem aktiven Plan. Bestätigen Sie die Sicherheitsabfrage.

6 Schüler-App generieren — Das Herzstück

Das Besondere: Mit einem Klick erstellen Sie eine fertige, eigenständige HTML-Datei, die Ihre Schüler einfach im Browser öffnen. Sie enthält genau die Module, die Sie ausgewählt haben — mit Fortschrittsverfolgung, Deadline-Countdown und persönlicher Begrüßung.

Schritt für Schritt

Module auswählen

Wählen Sie im Dashboard die gewünschten Module aus (siehe Kapitel 4).

Klicken Sie auf «Schüler-App»

Der grüne Button in der Kopfzeile. Es öffnet sich ein Konfigurationsdialog.

Klassenname eingeben

Z.B. «Klasse 7a», «MINT-AG» oder «Nachhilfe Max». Dieser Name erscheint als Überschrift in der Schüler-App.

Begrüßungstext schreiben

Eine persönliche Nachricht an Ihre Schüler. Z.B.:

Hallo liebe 7a!

Hier findet ihr eure Lernmodule für diese Woche.
Arbeitet die Module der Reihe nach durch und versucht
möglichst alle Quizze zu bestehen.

Viel Erfolg! Euer Herr Müller
Abgabedatum setzen (optional)

Wählen Sie ein Datum. In der Schüler-App erscheint dann ein Countdown: «Abgabe in 5 Tagen». Bei weniger als 3 Tagen wird die Anzeige rot.

Theme wählen

Wählen Sie ein Design für die Schüler-App (z.B. Comic für jüngere, Cyber für ältere Schüler). Die Schüler können später selbst zwischen Dark und Light umschalten.

App herunterladen

Klicken Sie auf «App herunterladen». Eine HTML-Datei wird automatisch heruntergeladen, z.B. Lernplan_Klasse_7a.html.

Verteilungswege:

  • USB-Stick: Gesamten Ordner auf USB-Stick kopieren und weitergeben.
  • Schulcloud / Moodle: Als ZIP-Datei hochladen. Schüler laden es herunter und entpacken es.
  • E-Mail: ZIP-Datei als Anhang senden.
  • Schul-Netzwerk: Auf ein gemeinsames Netzlaufwerk kopieren.

7 Schüler-Verwaltung — Presets, Schüler & Batch-Export

NEU: Die Schüler-Verwaltung ermöglicht es Ihnen, für jeden Schüler einen individuellen, personalisierten Lernplan zu erstellen — mit eigener Modul-Auswahl, persönlicher Nachricht und Deadline. Alle Lernpläne können auf einmal als ZIP-Datei heruntergeladen werden.

Schritt 1: Modul-Presets erstellen

Ein Preset ist eine gespeicherte Modul-Zusammenstellung, die Sie mehreren Schülern zuweisen können.

Module im Dashboard auswählen

Wählen Sie im Dashboard die gewünschten Module aus (z.B. alle Mathe-Module für Nachhilfe).

Zur Schüler-Ansicht wechseln

Klicken Sie in der Seitenleiste auf «Schüler».

Preset speichern

Klicken Sie auf «+ Aktuelle Auswahl als Preset speichern» und geben Sie einen Namen ein, z.B. «Mathe-Nachhilfe» oder «MINT-Basics».

Tipp: Erstellen Sie verschiedene Presets für verschiedene Förder-Szenarien: «Deutsch-Förderung», «Mathe-Vertiefung», «Komplettpaket Kl. 7». Jedes Preset kann beliebig vielen Schülern zugewiesen werden.
Preset-AktionBeschreibung
⬇ LadenLädt die Module des Presets zurück ins Dashboard (z.B. zum Bearbeiten).
✏️ UmbenennenÄndert den Namen des Presets.
✖ LöschenEntfernt das Preset. Schüler, die dieses Preset hatten, zeigen «(gelöscht)».

Schritt 2: Schüler anlegen

Schüler hinzufügen

Klicken Sie auf «+ Schüler hinzufügen». Es öffnet sich ein Formular.

Daten ausfüllen

Geben Sie den Namen ein, wählen Sie ein Preset aus, setzen Sie optional ein Abgabedatum und schreiben Sie eine persönliche Nachricht.

Speichern

Klicken Sie auf «Speichern». Der Schüler erscheint in der Liste mit Avatar-Initialen, Preset-Name und Details.

Schritt 3: Batch-Export

Sobald mindestens ein Schüler vorhanden ist, erscheint der Batch-Export-Bereich:

Basis-URL angeben (optional)

Wenn die Schüler die Lern-Apps per «Starten»-Button öffnen sollen, geben Sie hier den Pfad zum Schulbildung-Ordner an. Wird automatisch erkannt, wenn Sie das Dashboard aus dem Ordner öffnen.

Theme wählen

Wählen Sie ein Design für alle generierten Schüler-Apps.

ZIP herunterladen

Klicken Sie auf «Alle generieren & ZIP herunterladen». Alle Schüler-Apps werden in eine ZIP-Datei gepackt und heruntergeladen. Jede Datei heißt z.B. Lernplan_Dieter_Mueller.html.

Alternativ: Mit «Einzeln herunterladen» wird jede Schüler-App einzeln als HTML-Datei heruntergeladen (mit kurzer Verzögerung zwischen den Downloads).

Beispiel-Workflow

Szenario: Sie haben 3 Schüler mit unterschiedlichen Bedürfnissen:

1. Dieter — Probleme in Mathe → Preset «Mathe-Nachhilfe» (BruchProfi + AlgebraMeister)
2. Hans — Schwach in Deutsch → Preset «Deutsch-Förderung» (SchreibProfi + LeseHeld)
3. Wilfried — Will alles lernen → Preset «MINT-Komplett» (Mathe + Physik + Chemie + Bio)

➔ Ein Klick auf «Alle als ZIP generieren» → 3 personalisierte HTML-Dateien in einer ZIP-Datei. Jeder Schüler öffnet seine eigene Datei und sieht nur seine Module.

8 Fortschritt der Schüler verfolgen

NEU: Sie können jetzt den Lernfortschritt Ihrer Schüler direkt im Dashboard sehen — ohne Internet, ohne Anmeldung, ohne Server. Die Schüler speichern ihren Fortschritt als kleine Textdatei und geben sie Ihnen weiter.

So funktioniert es

Das Fortschritts-Tracking arbeitet in 3 einfachen Schritten:

📚
1. Schüler lernt
Der Schüler öffnet seine Lernplan-App und arbeitet Module ab. Der Fortschritt wird automatisch gespeichert.
💾
2. Schüler speichert
Ein Klick auf «Fortschritt speichern» erzeugt eine kleine Textdatei, die der Schüler an den Lehrer weitergibt.
📂
3. Lehrer lädt Datei
Im Dashboard klicken Sie auf «Datei laden», wählen die Datei aus — und sehen sofort den Fortschritt.
📊
4. Übersicht
Eine Karte zeigt: Name, erledigte Module, Quiz-Ergebnisse und Gesamt-Fortschritt in Prozent.

Anleitung für den Schüler

Geben Sie Ihren Schülern diese einfache Anleitung:

Öffne deine Lernplan-App

Das ist die HTML-Datei, die du von deinem Lehrer bekommen hast (z.B. Lernplan_Max.html).

Arbeite deine Module ab

Klicke auf «▶ Starten» bei einem Modul. Bearbeite das Quiz (mindestens 3 von 5 richtig) und die Boss-Aufgabe (alle 3 richtig). Erledigte Module bekommen ein ✓ Häkchen.

Klicke auf «💾 Fortschritt speichern»

Am Ende der Modulliste findest du diesen Button. Es wird eine kleine Textdatei heruntergeladen, z.B. Fortschritt_Max_Mueller_2026-03-09.txt.

Gib die Datei deinem Lehrer

Per USB-Stick, E-Mail, Schulcloud oder einfach ausdrucken — der Lehrer braucht nur diese eine kleine Datei.

10 Export, Import & Drucken

Lehrplan exportieren (JSON)

Klicken Sie auf «Export» in der Kopfzeile. Eine JSON-Datei wird heruntergeladen (z.B. lehrplan-2026-03-07.json). Diese Datei enthält:

  • Alle ausgewählten Module
  • Alle Wochenpläne
  • Ihr Theme und Ihre Einstellungen

Lehrplan importieren

Klicken Sie auf «Import» und wählen Sie eine zuvor exportierte JSON-Datei. Alle Einstellungen werden wiederhergestellt.

Anwendungsfall: Exportieren Sie Ihren Lehrplan am Schul-PC und importieren Sie ihn zu Hause auf Ihrem Laptop. Oder teilen Sie die JSON-Datei mit Kolleg:innen, damit diese denselben Plan verwenden können.

Wochenplan drucken

Klicken Sie auf «Drucken». Der Wochenplaner wird druckoptimiert dargestellt. Ideal als Aushang im Klassenzimmer oder als Handout für Eltern.

13 Häufige Fragen (FAQ)

Brauche ich Internet, um den LehrplanManager zu nutzen?
Nein. Der LehrplanManager und alle Lern-Apps funktionieren vollständig offline. Sie brauchen nur einen Browser.
Wo werden meine Daten gespeichert?
Ausschließlich lokal in Ihrem Browser (LocalStorage). Keine Daten verlassen Ihren Computer. Es gibt keinen Server, kein Tracking, keine Cookies.
Kann ich den LehrplanManager auf mehreren Geräten nutzen?
Ja, nutzen Sie dafür die Export/Import-Funktion. Exportieren Sie Ihren Plan als JSON-Datei und importieren Sie sie auf dem anderen Gerät.
Muss ich etwas installieren?
Nein. Doppelklicken Sie einfach auf Dashboard/index.html — die Datei öffnet sich in Ihrem Standard-Browser.
Funktioniert die Schüler-App auf dem Handy?
Ja. Sowohl der LehrplanManager als auch die Schüler-App und alle Lern-Apps sind vollständig responsive und funktionieren auf PC, Tablet und Smartphone.
Warum funktionieren die Links in der Schüler-App nicht?
Die Schüler-App enthält relative Links zu den Lern-Apps. Damit diese funktionieren, muss die HTML-Datei im Dashboard-Ordner liegen — oder der gesamte Schulbildung-Ordner muss mitgegeben werden. Verteilen Sie am besten alles zusammen als ZIP.
Können die Schüler die Antworten sehen?
Nein. Alle Quiz-Antworten in den Lern-Apps sind XOR-verschlüsselt. Die korrekten Antworten stehen nicht im Klartext im Quellcode.
Kann ich die Auswahl später ändern?
Ja. Ihre Auswahl wird automatisch im Browser gespeichert. Sie können jederzeit Module hinzufügen oder entfernen und eine neue Schüler-App generieren.
Wie viele Module sollte ich pro Woche zuweisen?
Als Richtwert: 2–3 Module pro Tag (50–75 Min.). Das hängt natürlich vom Fach, der Klassenstufe und der verfügbaren Zeit ab. Der Wochenplaner zeigt Ihnen die Gesamtzeit pro Tag an.
Kann ich mehreren Schülern unterschiedliche Module zuweisen?
Ja! Erstellen Sie verschiedene Presets (z.B. «Mathe-Nachhilfe», «Deutsch-Förderung») und weisen Sie jedem Schüler das passende Preset zu. Mit dem Batch-Export generieren Sie alle personalisierten Lernpläne auf einmal als ZIP.
Wie funktioniert die automatische Fortschrittserkennung?
Die Schüler-App liest alle 3 Sekunden den Lernfortschritt aus dem LocalStorage des Browsers. Wenn der Schüler ein Modul in einer Lern-App abschließt (Quiz bestanden), wird dies automatisch in der Schüler-App als «erledigt» markiert. Voraussetzung: Schüler-App und Lern-Apps laufen im selben Browser.
Was sind Presets und warum brauche ich sie?
Ein Preset ist eine gespeicherte Modul-Zusammenstellung. Anstatt für jeden Schüler einzeln Module auszuwählen, erstellen Sie einmal ein Preset und weisen es beliebig vielen Schülern zu. Das spart bei großen Klassen enorm viel Zeit.
Ist das DSGVO-konform?
Ja, vollständig. Keine personenbezogenen Daten werden erhoben, verarbeitet oder übertragen. Keine Cookies, keine Analyse-Tools, keine externen Verbindungen. Jede App enthält ein Impressum und eine Datenschutzerklärung.
